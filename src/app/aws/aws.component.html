About

Amazon Web Services (AWS) is the world's most comprehensive and broadly adopted cloud platform, offering over 200 fully featured services from data centers globally. Millions of customers—including the fastest-growing startups, largest enterprises, and leading government agencies—are using AWS to lower costs, become more agile, and innovate faster.

Objective

The striving motto of this workshop is to provide a multi-faceted view and learnings of “ AWS Solution Architect” for the aspirants

Glimpses Of The Workshop

Session I- Introduction to cloud computing

Session II- Identity & Access Management

Session III: Virtual Private Cloud

Session IV- Elastic IP Addresses

Session V: Elastic Cloud Compute (EC2)

Session VI- CloudFormation
