About

Data science is constantly evolving, with new techniques and tools being developed all the time. Staying up to date with the latest developments is essential for data scientists who want to stay competitive in the field.

Data has real value and every information extracted from it can make or break businesses. Being adept in Decision Science therefore, comes with its own powers. No data is clean but most is useful. Data science is all about asking the correct questions and analyzing the raw data. Data Analytics is the future and future is NOW!!

Objective

The striving motto of this workshop is to provide a multi-faceted view and learnings of “Data Science” for the aspirants. The potent takeaways include -

  • An Overall understanding of Data Science tools,technologies and its applications.
  • Learnings about Data analysis and Statistical analysis.
  • A complete overview on machine learning algorithms and data visualization.
  • The major benefits and real-time applications into limelight.

Glimpses Of The Workshop

Session I : Introduction to Data Science

Session II : Data Wrangling and Exploratory Data Analysis

Session III : Statistical Analysis and Hypothesis Testing

Session IV : Machine Learning Algorithms

Session V : Data Visualization and Communication

Session VI : Capstone Project
